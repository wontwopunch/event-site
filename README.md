# SignalDecode — 기업 랜딩 페이지

SignalDecode 브랜드를 위한 정적 기업 웹사이트 템플릿입니다.

## 프로젝트 개요

- **목표**: 기업용 IT·디지털 전환 메시지를 담은 반응형 랜딩 페이지
- **스타일**: 미니멀 흑백 + 블루 포인트 (#2563EB)
- **반응형**: 데스크톱 우선, 태블릿/모바일 지원

## 파일 구조

```
index.html
css/style.css
js/main.js
images/ (임시: [Lorem Picsum](https://picsum.photos)에서 내려받은 실제 JPG)
```

## 기술 스택

- HTML5, CSS3, Vanilla JavaScript
- Font Awesome 6 (CDN), Google Fonts (Pretendard / Inter)
